# House Price Dataset - Data Dictionary

This file describes all columns in the house dataset, their type, and why they are important for analysis.


 Column Name  | Meaning                  | Type    | Why Important                        |
| ------------ | ------------------------ | ------- | ------------------------------------ |
| SquareFeet   | House area in sqft       | Numeric | Larger houses usually cost more      |
| Bedrooms     | Number of bedrooms       | Numeric | Influences price                     |
| Bathrooms    | Number of bathrooms      | Numeric | Influences price                     |
| Neighborhood | Area / location          | Text    | Urban/Suburb/Rural affects value     |
| YearBuilt    | Year the house was built | Numeric | Can calculate house age or house age |
| Price        | Sale price               | Numeric | Target value for analysis            |
